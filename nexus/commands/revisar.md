Use o agente `nexus` para revisar o manuscrito fornecido. Contexto adicional: $ARGUMENTS

O NEXUS deve:
1. Acionar `revisor-pares` + `analista-critico` em paralelo para avaliação independente
2. Consolidar os problemas encontrados por criticidade (fatais → maiores → menores)
3. Acionar `redator-cientifico` para reescrever as seções problemáticas
4. Entregar: versão melhorada + relatório de mudanças + estratégia de submissão

Se `--periodico` não for informado, perguntar o periódico alvo antes de iniciar.
Solicitar ao usuário que cole o rascunho do manuscrito se ainda não tiver sido fornecido.
