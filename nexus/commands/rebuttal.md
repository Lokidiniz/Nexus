Use o agente `nexus` para montar a resposta aos revisores. Contexto: $ARGUMENTS

O NEXUS deve:
1. Acionar `revisor-pares` para classificar cada comentário (válido / parcialmente válido / inválido) e definir estratégia de resposta
2. Acionar `analista-critico` em paralelo para avaliar quais críticas requerem experimentos adicionais
3. Acionar `redator-cientifico` para redigir as respostas ponto a ponto e reescrever as seções alteradas
4. Entregar pacote completo: carta de resubmissão + respostas formatadas + manuscrito revisado com destaques

Solicitar ao usuário que cole os comentários dos revisores se ainda não fornecidos.
Perguntar o periódico alvo para calibrar o tom das respostas.
